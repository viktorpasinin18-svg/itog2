# Training Planner (План тренировок)

## Автор
Пашинин Виктор

## Описание

GUI-приложение на Python + Tkinter для планирования тренировок. Поддерживает добавление записей, фильтрацию по типу и дате, сохранение/загрузку в JSON.

## Функции
- Форма ввода: дата (YYYY-MM-DD), тип, длительность
- Таблица Treeview с сортировкой
- Фильтры: Combobox по типу, Entry по дате
- Валидация: формат даты, положительная длительность
- JSON: workouts.json (автосоздание)
- Responsive дизайн, скроллбары

## Установка и запуск
1. `pip install tkinter` (обычно предустановлен)
2. Сохраните `training_planner.py`
3. `python training_planner.py`

## Примеры использования
1. **Добавить:** `2026-05-04`, `Бег`, `45` → кнопка "Добавить"
2. **Фильтр типа:** Выберите "Бег" → "Применить"
3. **Фильтр даты:** `2026-05-04` → таблица только за дату
4. **Сохранить:** Автоматически в `workouts.json`
5. **Тест ошибок:**
   - Дата "abc" → ошибка
   - Длительность "-5" → ошибка
   - Пустые поля → ошибка
